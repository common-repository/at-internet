AT Internet official extension, allowing website tracking

== Description ==

AT Internet SmartTag allows you to automatically track your Wordpress website with [AT Internet](http://atinternet.com "AT Internet") solution.

== Installation ==

Installing "AT Internet SmartTag" can be done either by searching for "AT Internet SmartTag" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
